# Shape Your Music

A web application for drawing music.

This is V3, a new and improved React implementation of [V2](https://github.com/ejarzo/sym_v2), which was written in vanilla JS/jQuery.

## Project Set Up

* Node >= (v 4.0)

```bash
  $ npm install
```

### To run tests

```bash
  $ npm test
```

or

```bash
  $ npm test_watch
```

### To run project

```bash
  $ npm start
```
